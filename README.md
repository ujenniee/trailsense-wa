#trailsense-wa

trailsense is a hiking recommendation platform that helps Washington hikers identify the best day to hike specific trails using:

- Weather forecasts
- Air quality data
- Trail conditions
- Crowd predictions

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS

Backend
- FastAPI
- PostgreSQL

Data & ML
- Pandas
- Scikit-learn
